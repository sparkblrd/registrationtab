🔐 Flask Authentication System

A simple authentication system built with Python and Flask.

The project demonstrates the implementation of a complete user authentication flow, including registration, login, password hashing, and SQLite database integration.

## 🚀 Features

- User registration
- User login
- Password hashing with bcrypt
- SQLite database
- Duplicate username validation
- Login validation
- Responsive modern UI
- Glassmorphism design
- Flask backend


## 🛠 Tech Stack

### Backend
- Python 3
- Flask
- SQLite3
- bcrypt

### Frontend
- HTML5
- CSS3
- Boxicons

### Tools
- Git
- GitHub
- VS Code


## ⚙️ Installation

  Clone the repository

  Install dependencies ( // pip install flask bcrypt )

  Run the application

  Open in browser


## ⏱ Development Time

Approximately **4 hours**
